Coding and Computer Skills

In coding classes, students learn using project-based learning methods to learn real-world applications of each skill taught. Our program begins at the beginning stages of each language and works all the way up to advanced web development, emphasizing strength in the basics for better understanding and stronger foundational skills to build upon when learning advanced concepts.

Students can choose to learn just HTML and CSS, or expand their skills to HTML, CSS, Bootstrap 5 and basic JavaScript skills. We also offer basic Python Skills courses.
